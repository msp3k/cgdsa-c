void* mem_realloc ( void *  ptr,
size_t  size 
)

(Re)allocate size bytes of memory

Arguments:

  • void * ptr – A pointer to a location in memory
  • size_t size – Total number of bytes requested

Return value:

  • Succcess: void * – pointer to the head of the memory segment (which may have changed due to (re)allocation)
  • Failure: NULL

NOTES:

  • If ptr is NULL, then mem_realloc() acts like mem_alloc()
  • Ptr need not be the head of the memory segment, but may be a pointer to a location within a memory segment
