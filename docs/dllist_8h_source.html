225 /** Discern if two lists are identical
226  *
227  * Returns:
228  * - Success: 1 -- the lists are identical
229  * - Failure: 0 -- the lists differ
230  *
231  * Notes:

237  * - If one list is NULL and the other non-NULL, then return 0
