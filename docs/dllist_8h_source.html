223 /** Discern if two lists are identical
224  *
225  * Returns:
226  * - Success: 1 -- the lists are identical
227  * - Failure: 0 -- the lists differ
228  *
229  * Notes:

235  * - If one list is NULL and the other non-NULL, then return 0
