typedef struct __rbnode_t rbnode_t

Red-Black Trees.

rbtree_t Red-Black Tree A data structure used for a single node in the tree. It contains a void * pointer to a key, and a void * pointer to an associated value.

typedef struct __rbtree_t rbtree_t

A data structure used for the tree.

It contains a pointer to the root rbnode_t, a pointer to a compare function, a pointer to a free function, and a size_t value containing the number of nodes in the tree.
